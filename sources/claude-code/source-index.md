# Claude Code Source Index

## Purpose

This file tracks Claude Code related sources, their credibility, how they are archived locally, and how they may support later research.

## Credibility scoring

- A = primary / highly credible
- B = secondary / useful but needs cross-checking
- C = weak signal / discussion only

## Archive modes

- `archive-full` = store a near-complete markdown archive locally
- `archive-summary` = store a condensed markdown summary locally
- `extract-only` = keep only key extracted points / evidence
- `link-only` = keep only the external link and metadata

- video
